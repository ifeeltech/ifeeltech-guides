# Unsplash Image Guide

## Quick Reference for Finding Featured Images

### How to Get Unsplash Images

1. **Visit** [unsplash.com](https://unsplash.com)
2. **Search** for relevant keywords (e.g., "cybersecurity", "technology", "business security")
3. **Click** on an image you like
4. **Copy** the photo URL
5. **Format** it with parameters: `?w=1200&h=600&fit=crop`

### Example URL Format

```
https://images.unsplash.com/photo-[photo-id]?w=1200&h=600&fit=crop
```

### Image Specifications

- **Recommended size:** 1200x600px (2:1 aspect ratio)
- **Format:** Use `?w=1200&h=600&fit=crop` parameters
- **Credit:** Always add `image_credit: "Unsplash"` in frontmatter

### Frontmatter Example

```yaml
---
layout: default
title: "Your Article Title"
description: "Your description"
date: 2025-11-11
tags: [tag1, tag2]
featured_image: "https://images.unsplash.com/photo-1633265486064-086b219458ec?w=1200&h=600&fit=crop"
image_credit: "Unsplash"
---
```

### Tips for Selecting Images

✅ **Do:**
- Choose professional, high-quality images
- Select images that relate to the article topic
- Use abstract tech imagery for broad topics
- Prefer images with good contrast and clarity

❌ **Don't:**
- Use images with text overlays
- Choose overly busy or distracting images
- Select images with recognizable people (privacy concerns)
- Use low-resolution or pixelated images
